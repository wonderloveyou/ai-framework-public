# Platform Formats

Use exact platform requirements when the user provides them. Otherwise choose conservative defaults.

## Social Carousels

| Platform | Default | Also Useful | Notes |
|---|---:|---:|---|
| Instagram feed carousel | 1080x1350 | 1080x1080 | 4:5 gives more feed space. |
| Instagram square | 1080x1080 | 1080x1350 | Use when brand already has square templates. |
| LinkedIn carousel/document style | 1080x1350 or 1200x1500 | 1200x1200 | Optimize for mobile reading. |
| VK/TG square cards | 1920x1920 or 1080x1080 | 1080x1350 | Match the publishing habit. |
| X/Twitter image thread | 1600x900 | 1080x1080 | Keep text large. |

## Article Images

| Use | Default | Notes |
|---|---:|---|
| Article cover/header | 1920x1080 | Keep important text/object in safe central zone. |
| In-article explainer | 1920x1080 | Use consistent style across the article. |
| Wide preview card | 1200x628 | Only if target platform requires it. |

## Slide Count

- Short expert carousel: 4-6 slides.
- Educational carousel: 6-9 slides.
- Case study: 5-8 slides.
- Product/tutorial: 7-10 slides.

Avoid padding. A 5-slide clear carousel beats a 10-slide diluted one.

## Safe Zones

- Keep critical text away from edges.
- For 1920x1080, keep core content inside roughly `90px` horizontal and `100px` vertical margins.
- For mobile-first formats, avoid text below the lowest 8-10% if platform UI may cover it.
