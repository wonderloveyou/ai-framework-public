# B2B Content Marketing Skill

## Description
Complete B2B content marketing playbook: content formats, post templates, distribution channels, content planning, and lead magnets. Based on analysis of Sally AI, Roistat, Нескучные Финансы, and methods from "Сделаем" agency.

## When to Use
- Creating B2B content plan
- Writing specific post types (cases, guides, offers, etc.)
- Setting up distribution across channels
- Designing lead magnets and email sequences
- Planning Telegram channel as B2B showcase

## Process

### Step 1: Choose Content Format
Select from 10+ proven formats:
1. Multi-part guides (оглавление → части → анонс следующей)
2. Numbered lists (1️⃣ 2️⃣ 3️⃣ with "На выходе:" per step)
3. Message/copy breakdowns (пример → разбор → CTA)
4. Case studies (клиент → задача → действия → цифры → выводы)
5. Insight/opinion posts (2-4 paragraphs, one clear point)
6. Qualification posts ("Вам не нужен X, если...")
7. Offer posts (заголовок с цифрой → критерии → варианты → кейсы)
8. Event announcements (дата, время, спикеры, программа, регистрация)
9. Personal/community (живой эксперт, доверие)
10. Cross-promo (гости, подкасты, коллаборации)

### Step 2: Apply Post Structure Rules
- Bold заголовок + 1 эмодзи по смыслу
- Подзаголовки bold с цифрами
- Разделитель "———" перед CTA или подписью
- Подпись канала: "—— @nick — канал про [тема]"
- Внутренние ссылки на предыдущие посты
- Ссылки на продукт — в разборах и кейсах, не в каждом посте

### Step 3: Set CTA Type
- Soft CTA (educational posts): "напишите в комментарии", "маякните 🔥", "в следующем посте..."
- Hard CTA (offers/cases): "пишите в лс @nick", "контакт — @nick"
- Event CTA: "регистрируйтесь", "добавьте событие"
- Virality: "перешлите знакомым", "поделитесь с командами"

### Step 4: Plan Content Calendar
Frequency: 2-4 posts/week

Weekly rotation (Roistat-style):
- Mon: Educational (guide, list, methodology)
- Tue: Case study
- Wed: Webinar/Event
- Thu: Tips/Lifehacks
- Fri: News/Trends
- Sat: Light educational
- Sun: Rest or repost

### Step 5: Choose Distribution Channel
- Telegram: showcase, cases, direct CTA — PRIMARY
- LinkedIn: 2-4 posts/week, 70/20/10 ratio (edu/engage/promo) — TIER-1
- Email: drip sequences, lead magnets
- YouTube: low-effort, 1 video/2 weeks
- SEO: product pages + 2-3 articles on external platforms
- Reddit: native value-first, <10% self-promo
- Communities: Pavilion, CMO Coffee Talk, RevGenius

### Step 6: Add Hashtag System
Start with 5-7 рубрик:
- #полезное — educational
- #кейс — case studies  
- #мнение — expert opinion
- #инструменты — tools & services
- #дайджест — weekly digest
- #событие — events
- #команда — team, behind the scenes

## Pre-Publication Checklist
- [ ] Bold заголовок + 1 эмодзи
- [ ] Есть "на выходе" / цифры / выводы (where relevant)
- [ ] Разделитель + CTA/подпись
- [ ] В сериях — ссылка на предыдущую/следующую часть
- [ ] В офферах — ссылки на 2-4 кейса + контакт
- [ ] Нет воды: отвечает на "что именно" / "как" / "сколько"
